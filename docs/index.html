PACMAN HUNT

O clássico arcade reinventado em multiplayer

O QUE É PACMAN HUNT?

Uma nova experiência multiplayer onde você controla os fantasmas e caça o PacMan!

COMO FUNCIONA?

01

Cada jogador que entrar no jogo receberá um fantasma para controlar

02

O PacMan começará a se mover automaticamente pelo labirinto

03

Os fantasmas devem se reunir para capturá-lo e vencer

04

Se o PacMan comer todos os Pac Dots, ele vence a partida
